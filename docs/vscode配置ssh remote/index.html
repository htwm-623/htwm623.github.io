vscode配置ssh remote

安装 vscode 插件 remote-ssh​

Remote - SSH

使用 ssh 密钥登录远程服务器​

因为使用密码登录下次登陆时需要重新输入密码

  1. 使用 ssh-keygen 生成密钥

生成文件如下

  1. 使用cat命令将公钥文件，添加到远程主机authorized_keys 文件中，默认目录 ~/.ssh/

  1. vscode 添加配置文件

一般选择自己用户名下的配置文件即可 配置文件内容 

Host 120.24.44.188
HostName 120.24.44.188
User htwm623
Port 24247
IdentityFile "C:\Users\htwm6\.ssh\120_24_44_188_id_rsa"
  1. 点击主机名称右边图标即可连接
